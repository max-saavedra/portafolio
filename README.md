## Deployment (GitHub Pages)

1. Configure `vite.config.js`:

```js
export default {
  base: "/your-repo-name/"
}
```

2. Build the project:

```bash
npm run build
```

3. Deploy `dist/` using GitHub Pages:

* Go to repository → Settings → Pages
* Select branch: `gh-pages` or `/docs`
* Or use:

```bash
npm install gh-pages --save-dev
```

Add to `package.json`:

```json
"scripts": {
  "deploy": "gh-pages -d dist"
}
```

Then run:

```bash
npm run deploy
```
